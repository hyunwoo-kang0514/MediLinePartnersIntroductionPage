# MediLinePartners

Web introduction page for MediLinePartners

## About

MediLinePartners는 20년 연속 업계 1위 CRO 컨설팅 전문 기업입니다.

## Features

- **About Us**: 회사 소개 및 전문성
- **PV Service**: 전문적인 약물감시 서비스
- **RA Service**: 체계적인 인허가 컨설팅

## Technologies

- React 19
- Vite
- Framer Motion
- CSS3
